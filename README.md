# Music Web App

A modern Spotify-inspired music web application built with React.  
Users can discover music, create playlists, like songs, and enjoy a fully customizable experience with multiple themes.

## Features

- User Authentication (Sign up / Login)
- Play songs
- Search for songs
- Like / Unlike songs
- Create and manage playlists
- Upload and update profile picture
- Multiple theme support (selectable in user settings)

## Tech Stack

- **React** – UI Library
- **React Router** – Client-side routing
- **TanStack Query + Axios** – API Requests
- **Tailwind CSS** – Styling
- **shadcn/ui** – Reusable UI components
- **lucide-react** – Icons

## Installation

```bash
git clone git@github.com:MinPooneh7/music-web-app.git

cd music-web-app

npm install

npm run dev
```
